PRD Summary
Feature name: GestureNinja
Goal: Improve hand-eye coordination and sustained focus in dyslexic learners through gesture-controlled gameplay
Users: Students using NeuroLex, age 8+
Success metric: Session completion rate >70%, avg session >2 min
MVP scope:
[ ] Python WebSocket server (hand tracking)
[ ] Next.js /gesture-ninja page
[ ] Canvas game loop (spawn, arc, slice, lives, score)
[ ] Blade trail cursor
[ ] Connection status indicator
[ ] Basic post-game score screen
Out of scope for MVP:
Progress tracking integration (Phase 2)
Multiplayer
Mobile (needs different gesture backend)
